About this Web Application

This application gives you the real time mapped data of the Infected, Recovered,and Death population in Iowa. You can see your safety based on which county are you currently located in. You can zoom in the map for more clarity. Please click the button above to let this application find your location and double click to see you in the map below.

(Source: The map was created using geopandas and matplotlib in spyder and integrated in Django.)
